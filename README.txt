TRANSPORT - Delivery Management System
============================================

HOW TO START:
  Windows : Double-click  start.bat
  Mac/Linux: Run  ./start.sh  in terminal

Then open your browser to: http://localhost:8000

DEFAULT LOGIN:
  Username: admin
  Password: @dmin1234$

FIRST TIME SETUP:
  1. Go to Master Data > Parties and add your consigners and consignees
  2. Go to Master Data > Routes and add your routes
  3. Start creating vouchers!

DATA SAFETY:
  - Your data is stored in: kap_transport.db (keep this file safe)
  - Every time you start the app, an auto-backup is saved in the backups/ folder
  - You can also manually backup from the sidebar (Backup Data button)
  - To move to another computer, copy the entire folder including kap_transport.db

CHANGE PASSWORD:
  Go to http://localhost:8000/admin to manage users and change passwords.
